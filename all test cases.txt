🔷 SECTION A — Pure Grandparent Configurations
✅ Case 1 — Only Paternal Grandparents

Heirs:

PGF

PGM

Rule: Entire estate, male double female.

Distribution:
PGF = 2/3
PGM = 1/3

✅ Case 2 — Only Maternal Grandparents

Heirs:

MGF

MGM

Rule: Entire estate, equal.

MGF = 1/2
MGM = 1/2

✅ Case 3 — All Four Grandparents

Heirs:

PGF

PGM

MGF

MGM

Rule:
Maternal side = 1/3
Paternal side = 2/3

Final:
PGF = 4/9
PGM = 2/9
MGF = 1/6
MGM = 1/6

✅ Case 4 — PGF + PGM + Only MGF

Maternal side (1 person) = 1/3
Paternal side (2 persons) = 2/3

MGF = 1/3

Paternal (2:1 inside 2/3):
PGF = 4/9
PGM = 2/9

✅ Case 5 — PGF + PGM + Only MGM

Same as Case 4

MGM = 1/3

PGF = 4/9
PGM = 2/9

✅ Case 6 — MGF + MGM + Only PGF

Maternal group = 1/3 → equal

MGF = 1/6
MGM = 1/6

PGF alone = full 2/3

✅ Case 7 — MGF + MGM + Only PGM

Maternal group = 1/3 → equal

MGF = 1/6
MGM = 1/6

PGM alone = full 2/3

✅ Case 8 — Only PGF

No one else.

He gets entire estate via radd.

PGF = 100%

✅ Case 9 — Only MGF

MGF = 100% (radd)

✅ Case 10 — Only PGM

PGM = 100% (radd)

✅ Case 11 — Only MGM

MGM = 100% (radd)

🔷 SECTION B — Multiple Grandmothers (Blocking Rule)

Text says:

Closest blocks further.

✅ Case 12 — 3 Paternal Grandmothers

Only closest (mother of father) inherits.

She takes entire estate (radd).
Other two blocked.

✅ Case 13 — 2 Maternal Grandmothers

Only closest (mother of mother) inherits.
She takes entire estate.

✅ Case 14 — 3 PGM + 2 MGM (No other heir)

Maternal group = 1/3
Paternal group = 2/3

But only closest per line counts:

Closest PGM = 2/3
Closest MGM = 1/3

Others blocked.

🔷 SECTION C — Fixed 1/6 Grandmother Rule

From Hadith:

Prophet gave grandmother 1/6 even when her son alive.

✅ Case 15 — Father + PGM

PGM = 1/6
Father = 5/6

✅ Case 16 — Mother + MGM

MGM = 1/6
Mother = 5/6

✅ Case 17 — Father + Mother + One Grandmother

Grandmother = 1/6
Mother = 1/6
Father = remainder (2/3)

✅ Case 18 — Father + Two Grandmothers (PGM + MGM)

Each grandmother = 1/6
Father = remainder

🔷 SECTION D — Grandfather Treated as Brother
✅ Case 19 — PGF + 2 Maternal Brothers

Maternal brothers = 1/3
PGF = 2/3

✅ Case 20 — PGF + 1 Maternal Brother

Maternal brother = 1/6
PGF = 5/6

✅ Case 21 — MGF + 2 Paternal Brothers

MGF = 1/6
Paternal brothers = 5/6

🔷 SECTION E — Grandfather with Nephew

Text:

Nephew inherits what father would inherit.

✅ Case 22 — PGF + Nephew (son of brother)

Treat as if:
PGF + Brother

Divide as siblings 2:1

PGF = 2/3
Nephew = 1/3

✅ Case 23 — PGF + Multiple Nephews

They inherit what their father would get, divided among them.

Example:
If father share = 1/3
3 nephews divide equally inside that share.

🔷 SECTION F — Daughter Example from Text
✅ Case 24 — 3 PGM + 2 MGM + 1 Daughter

Closest PGM = 1/6
Closest MGM = 1/6
Daughter = remainder (2/3)

Others blocked.
No radd to grandmothers.

🔷 SECTION G — Grandfather + Grandmother + Brothers
✅ Case 25 — PGF + PGM + Full Brother

PGM = 1/6
Remaining = divided between PGF and brother equally (as siblings).

Remaining = 5/6

PGF = 5/12
Brother = 5/12
