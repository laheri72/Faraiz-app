PROJECT NAME:
Fatemi Wirasat Engine

PURPOSE:

Create a deterministic inheritance calculation engine based exclusively on Daim al-Islam Bab al-Faraiz jurisprudence.

This system must calculate inheritance shares and explain reasoning according to verified rules.

The system must:

- Identify heirs
- Apply eligibility rules
- Apply fixed shares
- Apply priority inheritance
- Apply return by kinship (Radd)
- Perform exact mathematical calculations
- Produce full explanation

This is NOT an AI-based fatwa generator.

This is a deterministic jurisprudence engine.

AI may assist only in UI or explanation generation.

All legal logic must come only from verified rules.
