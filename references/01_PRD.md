PRODUCT REQUIREMENTS DOCUMENT


GOAL

Build a web application that calculates Fatemi inheritance according to Daim al-Islam.

Users input:

- Estate value
- Heirs list

System outputs:

- Share of each heir
- Calculation steps
- Legal reasoning


PRIMARY FEATURES

Feature 1:

Heir Input Interface

User selects:

Son
Daughter
Father
Mother
Husband
Wife
Brother
Sister
Uncle
Cousin
Dhawu al-Arham


Feature 2:

Eligibility Processing

System removes invalid heirs.


Feature 3:

Fixed Share Calculation

Apply أهل الفرائض rules.


Feature 4:

Priority Distribution

Apply الأقرب فالأقرب logic.


Feature 5:

Radd Distribution

Return remainder.


Feature 6:

Exact Calculation

Fraction-based calculations.


Feature 7:

Explanation Output

Display rules used.


NON GOALS

No fatwa issuing.

No probabilistic AI decisions.

No rule guessing.


SUCCESS CRITERIA

Engine matches Daim al-Islam rules exactly.
